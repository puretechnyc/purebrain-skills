## Quick Start

```bash
# Install any skill in 30 seconds
git clone https://github.com/puretechnyc/purebrain-skills.git
cp -r purebrain-skills/skills/operations/build-your-brain ~/.claude/skills/
```

Then in Claude Code:

```
/skill build-your-brain
```

That's it. The skill is now available in every session.
